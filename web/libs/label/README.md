# 文档分析管理系统

基于React + Ant Design的事件标签标注Demo系统

## 安装和运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm start
```

### 3. 构建生产版本
```bash
npm run build
```

## 功能特性

- Excel文件上传解析
- 事件标注和导航
- 威胁/差错/UAS识别
- 训练主题自动映射
- 胜任力评估

## 使用说明

1. 点击"上传文档"选择Excel文件
2. 使用"上一条/下一条"浏览事件
3. 在各模块中进行标注
4. 点击"保存标注"保存到内存
5. 点击"提交审核"完成标注

## 技术栈

- React 18
- Ant Design 5
- xlsx (Excel解析)
- dayjs (日期处理)
